详细说明：
==================================================================================
【功能】
  提供Windows环境下的一键式图形界面软件为客户自动安装WAMP及FTP环境并可以自动将客户
之前的虚拟主机（Apache+PHP+MySQL）的数据库、文件和账号信息同步到阿里云ECS服务器，
一键完成整站迁移。

【特点】
  1.自动化搭建环境。WAMP环境的搭建采用开源的WampServer，软件自动选择WampServer当前
    的稳定版本，下载并自动安装；FTP服务器采用开源的FileZilla Server，软件自动选择
    FileZilla Server当前的稳定版本，下载并自动安装。如果服务器是64位架构，则优先考
    虑安装64位的服务器软件。
  2.自动化同步数据库。通过调用mysqldump命令自动远程备份原网站数据库、自动下载、自
    动导入到新服务器的MySQL数据库。当MySQL远程备份服务未开启时，自动转为使用PHP脚
    本备份数据库。
  3.自动化同步文件。通过内建的FTP客户端连接原服务器，自动下载原网站的全部文件到本
    地。独有的快速同步模式针对FTP的目录切换做了优化。
  4.展示同步状态。软件内置动态页面文件，提供同步的状态信息，用户可以在任意电脑通过
    浏览器查看阿里云ECS服务器当前的迁移状态。当全部同步工作完成后该动态页面会自动
    被原网站的首页替换。
  5.保留原有账号信息不变。本地MySQL数据库的数据库名、用户名和密码信息同原服务器相
    同，FTP服务器的用户名和密码同原服务器相同。

【使用】
  本软件是一款绿色软件，不需要安装。用户远程登录到阿里云ECS主机，下载运行本软件即
可进行一站式同步。用户在软件界面上设定一些参数后，软件自动开始下载安装必要软件、搭
建网站环境、进行原网站的数据迁移工作。一般情况下，网站迁移的数据同步的时间往往比较
长（由网站的规模决定），为了解决此问题，本软件提供了通过网站临时主页查看迁移进度的
功能，即用户在开始数据迁移后就可以退出登录远程桌面，之后随时通过“http://服务器IP
地址”查看迁移进度。当网站迁移结束后，在阿里云服务器上新网站的数据库、文件和账号信
息会和原服务器完全相同，新主页生效，至此完成了一次整站迁移。迁移后用户可以先通过阿
里云ECS服务器的IP地址测试新部署的网站，如果没有兼容性问题，只要更新域名的DNS记录指
向阿里云ECS服务器IP地址就可以让网站在阿里云上线。

【优势】
  自动完成、操作简单、功能全面。

【价值】
  由于目前主流的虚拟主机普遍采用Apache+PHP+MySQL技术，所以相信本软件可以帮助很多需
要将虚拟主机迁移到阿里云ECS的网站站长。鉴于国内的Windows用户比较广泛，本次发布的是
Windows版本的图形界面软件。

技术概览：
==================================================================================
应用程序使用了Delphi XE 3的FireMonkey技术以及Windows 8灰色主题，界面设计清爽大方，
操作方便，功能强大，为用户初次使用阿里云主机、整站迁移到阿里云主机提供一站式服务。
应用程序可以在Windows 8以外的Windows操作系统中正常运行。
